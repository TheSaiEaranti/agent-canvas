# Agent Canvas

> A visual canvas for composing and debugging AI agents.

Drag out boxes (prompts, tool calls, outputs), connect them with arrows, hit **Run**, and watch each box light up and fill with output as the agent executes — making AI agent runs visible and debuggable instead of a wall of scrolling text.

- **[SPEC.md](./SPEC.md)** — product & technical spec
- **[PLAN.md](./PLAN.md)** — milestone breakdown (M1–M4)
